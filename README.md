# AI-Powered Lead Management Automation System

## 📌 Overview

This project demonstrates how AI and automation can streamline customer enquiry handling in a medical aesthetics setting.

It automates the process of capturing, analysing, and responding to leads using GPT-powered AI via Zapier workflow automation tools.

---

## ⚙️ Workflow

1. User submits enquiry via Google Form
2. Data is stored in Google Sheets
3. Zapier detects new entry
4. GPT models (via Zapier AI – OpenAI) process the enquiry:
   - Generate a concise summary
   - Classify intent (e.g., Botox, fillers, skincare, general enquiry)
   - Create a professional, friendly response
5. Automated email is sent with AI insights

---

## 🛠️ Tech Stack

   - Python (optional, for API interaction or data processing/transformation to clean or manipulate form data before sending to AI)
   - Zapier (workflow automation)
   - GPT models (via Zapier AI – OpenAI)
   - Google Forms & Google Sheets
