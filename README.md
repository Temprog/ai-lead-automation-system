# AI-Powered Lead Management Automation System

## 📌 Overview
This project demonstrates how AI and automation can streamline customer enquiry handling in a medical aesthetics setting.

It is currently being developed to automate the process of capturing, analysing, and responding to leads using GPT and workflow automation tools.

---

## ⚙️ Planned Workflow

1. User submits enquiry via Google Form  
2. Data is stored in Google Sheets  
3. Automation tool (Zapier) detects new entry  
4. OpenAI GPT processes the enquiry:
   - Generates a summary  
   - Classifies intent  
   - Creates a professional response  
5. Automated email is generated with AI insights  

---

## 🛠️ Tech Stack

- Python (for API interaction)
- Zapier / Make (workflow automation)
- OpenAI API (GPT)
- Google Forms & Google Sheets
